# Phase 4 — Preprocessing

## تتعلم إيه؟

تتعلم إزاي نحول radiographs إلى شكل مناسب للنموذج:

```text
Image → resize → normalize → tensor
Mask → resize → binary → tensor
```

## تنفذ إيه؟

تعمل preprocessing function:

```python
IMG_SIZE = 256

def load_image(path):
    image = cv2.imread(path, cv2.IMREAD_GRAYSCALE)
    image = cv2.resize(image, (IMG_SIZE, IMG_SIZE))
    image = image / 255.0
    image = np.expand_dims(image, axis=-1)
    return image.astype(np.float32)

def load_mask(path):
    mask = cv2.imread(path, cv2.IMREAD_GRAYSCALE)
    mask = cv2.resize(mask, (IMG_SIZE, IMG_SIZE))
    mask = mask / 255.0
    mask = (mask > 0.5).astype(np.float32)
    mask = np.expand_dims(mask, axis=-1)
    return mask
```

## تفهم إيه؟

الـ model لا يرى “صورة” مثل الإنسان.
هو يرى numbers:

```text
0.0 → black
1.0 → white
between → gray
```

أما الـ mask فهو target:

```text
0 = background
1 = tooth
```

# Phase 8 — بناء U-Net Model

## تتعلم إيه؟

تتعلم كيف تبني architecture بنفسك بدل استخدام model جاهز.

## تنفذ إيه؟

```python
from tensorflow.keras import layers, models

def conv_block(inputs, filters):
    x = layers.Conv2D(filters, 3, padding="same", activation="relu")(inputs)
    x = layers.Conv2D(filters, 3, padding="same", activation="relu")(x)
    return x

def build_unet(input_shape=(256, 256, 1)):
    inputs = layers.Input(input_shape)

    c1 = conv_block(inputs, 64)
    p1 = layers.MaxPooling2D()(c1)

    c2 = conv_block(p1, 128)
    p2 = layers.MaxPooling2D()(c2)

    c3 = conv_block(p2, 256)
    p3 = layers.MaxPooling2D()(c3)

    c4 = conv_block(p3, 512)
    p4 = layers.MaxPooling2D()(c4)

    bn = conv_block(p4, 1024)

    u1 = layers.UpSampling2D()(bn)
    u1 = layers.Concatenate()([u1, c4])
    c5 = conv_block(u1, 512)

    u2 = layers.UpSampling2D()(c5)
    u2 = layers.Concatenate()([u2, c3])
    c6 = conv_block(u2, 256)

    u3 = layers.UpSampling2D()(c6)
    u3 = layers.Concatenate()([u3, c2])
    c7 = conv_block(u3, 128)

    u4 = layers.UpSampling2D()(c7)
    u4 = layers.Concatenate()([u4, c1])
    c8 = conv_block(u4, 64)

    outputs = layers.Conv2D(1, 1, activation="sigmoid")(c8)

    model = models.Model(inputs, outputs)
    return model
```

آخر layer فيه:


# Phase 9 — Loss Functions and Metrics

## تتعلم إيه؟

تتعلم ليه accuracy وحدها غير كافية.

في segmentation، الخلفية ممكن تكون كبيرة، فلو النموذج قال معظم الصورة background ممكن يجيب accuracy كويسة لكن segmentation سيء.

## تنفذ إيه؟

تستخدم:

```text
Dice coefficient
IoU
Binary crossentropy
Dice loss
```

مثال:

```python
def dice_coef(y_true, y_pred, smooth=1):
    y_true_f = tf.keras.backend.flatten(y_true)
    y_pred_f = tf.keras.backend.flatten(y_pred)
    intersection = tf.keras.backend.sum(y_true_f * y_pred_f)
    return (2. * intersection + smooth) / (
        tf.keras.backend.sum(y_true_f) + tf.keras.backend.sum(y_pred_f) + smooth
    )

def dice_loss(y_true, y_pred):
    return 1 - dice_coef(y_true, y_pred)

def bce_dice_loss(y_true, y_pred):
    return tf.keras.losses.binary_crossentropy(y_true, y_pred) + dice_loss(y_true, y_pred)
```
